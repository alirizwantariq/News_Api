
# News Articles

A website to read latest and popular news articles from all around the world


## API Reference

#### Get all items

```http
  GET https://newsapi.org/everything?
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
